📋 IMPLEMENTATION DETAILS
═══════════════════════════════════════════════════════════════════════

USER FLOW:
  1. User creates interview (+ Create Interview)
  2. Provides resume and/or job description
  3. System generates interview questions
  4. D-ID avatar created with greeting script
  5. Professional avatar displays in dialog
  6. Avatar speaks greeting to user
  7. Questions saved to database
  8. User redirected to interview session
