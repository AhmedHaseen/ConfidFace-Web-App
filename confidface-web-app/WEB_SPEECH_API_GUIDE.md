# 🎤 Free Interview Conversation System - Web Speech API

## Overview
This implementation uses **completely free browser APIs** for both text-to-speech and speech-to-text. No API keys, no paid services, no backend dependencies.

## Free Technologies Used

### 1. **Web Speech API (STT)** ✅ FREE
- **Browser native API** - Works in Chrome, Edge, Safari, Firefox
- **No API keys required**
- **No backend processing needed**
- **Converts speech directly to text in the browser**

### 2. **Web Speech Synthesis API (TTS)** ✅ FREE
- **Browser native API** - Works in all modern browsers
- **No API keys required**
- **No cost**
- **Multiple voice options available**

## How It Works

### Conversation Flow

```
┌─────────────────────────────────────────────────────────────────┐
│ 1. User clicks "Connect Call"                                  │
│    ↓                                                             │
│ 2. Bot greeting generated by Gemini (backend)                  │
│    ↓                                                             │
│ 3. Bot greeting SPOKEN using Web Speech API (FREE) 🔊          │
│    ↓                                                             │
│ 4. User clicks "Mute" to STOP speaking                         │
│    ↓                                                             │
│ 5. Web Speech API recognizes user's speech 🎤                 │
│    ↓                                                             │
│ 6. User's transcript added to conversation                     │
│    ↓                                                             │
│ 7. Gemini generates bot response                               │
│    ↓                                                             │
│ 8. Bot response SPOKEN using Web Speech API (FREE) 🔊          │
│    ↓                                                             │
│ 9. User can click "Unmute" to speak again                      │
└─────────────────────────────────────────────────────────────────┘
```

## Implementation Details

### Web Speech Utilities (`utils/webSpeechAPI.ts`)

```typescript
// Speech-to-Text (STT) Hook
const { startListening } = useSpeechToText();

// Usage:
startListening(
  (transcript) => { console.log("User said:", transcript); },
  (error) => { console.error("Error:", error); }
);

// Text-to-Speech (TTS) Hook  
const { speak, stop } = useTextToSpeech();

// Usage:
speak("Hello! How can I help?", () => {
  console.log("Speech finished!");
});
```

## Limitations & Workarounds

| Limitation | Workaround |
|-----------|-----------|
| Accuracy depends on microphone | Use quality microphone in quiet environment |
| Some browsers require HTTPS | Use localhost:3000 for development |
| Limited voice options | Web Speech API provides system voices |
| Language limited to browser setting | Can be changed in browser settings |

## Troubleshooting

### Speech Recognition Not Working
- Check microphone permissions
- Use HTTPS (or localhost for dev)
- Try a different browser
- Use a better quality microphone

### Text-to-Speech Not Playing
- Check browser volume
- Check system volume
- Try a different browser
- Clear browser cache

### No Microphone Detected
- Check system settings
- Verify microphone is connected
- Refresh the page
- Try a different browser
